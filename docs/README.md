# Search Plugin
Плагин для IntelliJ Idea, осуществляющий поиск выделенного текста в Google

## Автор: Абросимов Арсений Сергеевич M3216

## Описание
<!-- Plugin description -->
Данный плагин при нажатии комбинации клавиш Ctrl+Alt+G открывает поисковую строку с запросом в виде выделенного текста
<!-- Plugin description end -->
